## 1. next-intl 세팅 (권장)

```bash
pnpm add next-intl
```

```tsx
// middleware.ts
import createMiddleware from 'next-intl/middleware';

export default createMiddleware({
  locales: ['ko', 'en', 'ja'],
  defaultLocale: 'ko',
});

export const config = {
  matcher: ['/((?!api|_next|_vercel|.*\\..*).*)'],
};
```

## 2. 메시지 구조

```json
// messages/ko.json
{
  "common": {
    "submit": "제출",
    "cancel": "취소",
    "loading": "로딩 중..."
  },
  "dashboard": {
    "title": "대시보드",
    "greeting": "안녕하세요, {name}님!"
  },
  "products": {
    "count": "{count}개의 상품"
  }
}
```

### 사용

```tsx
import { useTranslations } from 'next-intl';

function Dashboard() {
  const t = useTranslations('dashboard');
  return <h1>{t('title')}</h1>;  // "대시보드"
}

// 매개변수
<p>{t('greeting', { name: user.name })}</p>  // "안녕하세요, 철수님!"

// 복수형
<p>{t('products.count', { count: 5 })}</p>  // "5개의 상품"
```

## 3. 날짜/시간/숫자 포맷

```tsx
import { useFormatter } from 'next-intl';

function Stats() {
  const format = useFormatter();

  return (
    <div>
      <p>{format.dateTime(new Date(), { dateStyle: 'long' })}</p>
      {/* ko: 2024년 3월 13일, en: March 13, 2024 */}

      <p>{format.number(1234567.89, { style: 'currency', currency: 'KRW' })}</p>
      {/* ko: ₩1,234,568, en: KRW 1,234,567.89 */}

      <p>{format.relativeTime(new Date('2024-03-10'))}</p>
      {/* ko: 3일 전, en: 3 days ago */}
    </div>
  );
}
```

## 4. 언어 전환

```tsx
import { usePathname, useRouter } from 'next/navigation';
import { useLocale } from 'next-intl';

function LanguageSwitcher() {
  const locale = useLocale();
  const router = useRouter();
  const pathname = usePathname();

  const switchLocale = (newLocale: string) => {
    const newPath = pathname.replace(`/${locale}`, `/${newLocale}`);
    router.push(newPath);
  };

  return (
    <select value={locale} onChange={e => switchLocale(e.target.value)}>
      <option value="ko">한국어</option>
      <option value="en">English</option>
      <option value="ja">日本語</option>
    </select>
  );
}
```

## 5. SEO

```tsx
// app/[locale]/layout.tsx
export function generateMetadata({ params }: { params: { locale: string } }) {
  return {
    title: {
      ko: '서비스명',
      en: 'Service Name',
    }[params.locale],
    alternates: {
      canonical: `https://example.com/${params.locale}`,
      languages: {
        'ko-KR': `https://example.com/ko`,
        'en-US': `https://example.com/en`,
      },
    },
  };
}
```

## ❌ 안티패턴

* **하드코딩된 문자열**: "제출" → `t('common.submit')`
* **날짜 포맷 직접 구현**: `new Date().toLocaleDateString()` → `format.dateTime()`
* **번역 키 중복**: `dashboard.title` / `dashboard_title` 혼용
* **서버/클라이언트 불일치**: 서버에서 `ko`, 클라이언트에서 `en` → hydration 에러
